# Session Todo
> Lightweight running checklist for the current work session. The agent writes the plan here before implementation and checks items off as it goes.

## How to Use
- **At session start:** Agent creates a new section with today's date and the planned work.
- **Before coding:** Write checkable items for each step.
- **During work:** Mark items `[x]` as they're completed.
- **At session end:** Add a review section with outcome and any follow-ups.
- **Between sessions:** This file persists. The agent reads it to resume where it left off.

## Format
```markdown
## YYYY-MM-DD — [Goal for today]

### Plan
- [ ] Step 1
- [ ] Step 2
- [ ] Step 3

### Review
- **Outcome:** [What was accomplished]
- **Blockers:** [What's still stuck]
- **Next:** [What to do next session]
```
